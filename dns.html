Domain Name System

Definitions

DNS:
Known as Domain Name System, is a system that associates website names with IP Addresses. Its designed to scale and provides the protocol that provides the framework for web browsing.

Examples

From the image above, DNS establishes a process of retrieving an IP Address for a specific domain name so that an internet resource can be retrieved. This procedure involves querying multiple servers until one server has it and passes it along. The server that sends that information caches it for later use.

Simply put: A computer uses the DNS to look up domain names and get the associated IP Address its using to connect to the destination on the internet.

Why are these useful?

With DNS, we are able to go through many websites because of the X many servers we have always running and retrieving the selected information we requested. Without it, 1 server may not be able to retrieve everything a user asks. It breaks through many limitations with the usage of many servers and caching.
