W3C, HTML and CSS

Definitions

W3C:
Standing for the World Wide Web Consortium, is an international community that helps develop web standards for HTML and CSS. It allows for the information to be accessible for everyone, faster and more enjoyable to use. The organization is led by Tim Berners Lee.

HTML:
Standing for Hyper Text Markup Language, it is a standard markup language that helps to create web pages. Basically describes the structure of a web page, where it consists of a series of elements known as tags.

CSS:
Standing for Cascading Style Sheets, it tells how a web document should look like by adding `style` to it. These style enhancements are related to color, font and page layouts etc.

Examples

Whenever a user visits a site, the combination of w3c, html and css allows for the content to be displayed properly and in an efficient manner. A website is solely composed of HTML and CSS through which are done through rules set up by W3C.

Why are these useful?

A website would not be a site for us if it displayed no content nor was it styled properly with different fonts, colors etc. Same can be said of W3C, which establishes the guidelines upon which HTML and CSS are built upon. The internet's popularity would not have reached today's levels if it weren't for the advancements in all 3 categories.
