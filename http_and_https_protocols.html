HTTP and HTTPS Protocols

Definitions

HTTP:
A client-server protocol that stands for Hyper Text Transfer Protocol, provides the means of any form of data exchanges on the web.

HTTPS:
The secure version of HTTP, Hyper Text Transfer Protocol Secure, ensures that any data exchanges done on the web is encrypted to ensure security of the dat transfer.
